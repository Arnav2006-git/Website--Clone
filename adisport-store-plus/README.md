# Adisport Store Plus (Adidas‑inspired UI, brand‑safe)

A more *similar* UX to major sportswear sites: home hero, mega nav links, PLP with filters/sort, PDP with gallery and sticky add-to-bag, simple cart placeholder.

## Quick start
```bash
npm install
npm run dev
```

## Edit data
- `src/data.js` for products
- `src` for pages and components

This is an educational demo. Do not use real brand assets or copy‑paste text from any site.
